## ✨ Descripción

**Legends-GameProject** es un videojuego RPG 2D donde el jugador explora un mundo lleno de misterios, personajes y desafíos.

Diseñado con enfoque en:
- 🌍 Exploración
- ⚔️ Combate  
- 🧠 Interacciones significativas
- 📈 Progresión del jugador
